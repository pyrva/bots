# PyRVA Bots

This repository houses the bots that help manage PyRVA. 

## Build Environment

Set up your environment

```
python -m venv .venv
source .venv/bin/activate
pip install flit
flit install -s
```

- [Flit](https://flit.readthedocs.io/)
- [pytest](https://docs.pytest.org/)
